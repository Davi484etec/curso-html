Como usar as tags em HTML.

Aprenda a usar as tags HTML de forma simples e prática.

O HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web.
Ele é usado para estruturar o conteúdo em parágrafos, listas, tabelas e imagens.

O que são as Tags?

As tags são os elementos fundamentais do HTML, que definem a estrutura e o conteúdo de uma página web. Elas são usadas para criar títulos, parágrafos, listas, links, imagens e muito mais.

As tags HTML são compostas por uma abertura e um fechamento, como por exemplo: <tag>Conteúdo</tag>. Algumas tags são auto-fechadas, como a tag de imagem: <img src="imagem.jpg" alt="Descrição da imagem" />.

As tags HTML podem ter atributos, que fornecem informações adicionais sobre o elemento. Por exemplo, a tag de link pode ter um atributo href que especifica o URL do link: <a href="https://www.exemplo.com">Clique aqui</a>.

As tags HTML são organizadas em uma hierarquia, onde algumas tags podem conter outras tags dentro delas. Por exemplo, a tag de parágrafo pode conter tags de formatação, como <b> para negrito ou <i> para itálico.

Exemplo de Tags HTML.

Veja abaixo um exemplo de como usar as tags HTML para criar uma página simples:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Título Principal</h1>
    <p>Este é um parágrafo de exemplo.</p>
    <a href="https://www.exemplo.com">Clique aqui</a>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>
    <p><img src="imagem.jpg" alt="Descrição da imagem"></p>
    <p>Texto em <b>negrito</b> e <i>itálico</i>.</p>
    <h2>Subtítulo</h2>
    <p>Mais conteúdo aqui.</p>
    <h3>Subtítulo Secundário</h3>
    <p>Conteúdo adicional.</p>
    <h4>Subtítulo Terciário</h4>
    <p>Conteúdo extra.</p>
    <h5>Subtítulo Quaternário</h5>
    <p>Conteúdo complementar.</p>
    <h6>Subtítulo Quinário</h6>
    <p>Conteúdo final.</p>
    <footer>
        <p>Rodapé da página</p>
    </footer>
</body>
</html>

Esse exemplo mostra como usar as tags HTML para criar uma estrutura básica de uma página web. Você pode personalizar o conteúdo e o estilo usando CSS.

Para mais informações sobre HTML, consulte a documentação oficial do W3C: HTML5 Specification.
